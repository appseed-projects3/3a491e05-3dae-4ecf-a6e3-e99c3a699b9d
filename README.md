### ðŸ‘‰ Start the Backend Server

> **Step 2** - Install dependencies using a `virtual environment`


```bash

$

$ (Powershell) $env:FLASK_ENV = "development"
```

> **Step 4** - Start the API server (development mode)

```bash
$ flask run
```
